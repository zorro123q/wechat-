# 图片资源说明

本目录存放小程序所需的图片资源。

## 需要准备的图片

### TabBar 图标 (建议尺寸: 81px * 81px)

- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `mall.png` - 商城图标（未选中）
- `mall-active.png` - 商城图标（选中）
- `profile.png` - 我的图标（未选中）
- `profile-active.png` - 我的图标（选中）

### 其他图片

- `logo.png` - 学院Logo (建议尺寸: 160px * 160px)
- `default-avatar.png` - 默认头像 (建议尺寸: 200px * 200px)
- `default-goods.png` - 默认商品图 (建议尺寸: 400px * 400px)

## 临时方案

在正式图片准备好之前，可以：

1. 使用纯色占位图
2. 使用在线图标库
3. 使用 emoji 代替（代码中已使用 emoji 作为临时方案）

## 图片格式要求

- 格式：PNG 或 JPG
- 大小：单个文件不超过 200KB
- 命名：使用小写字母和连字符
