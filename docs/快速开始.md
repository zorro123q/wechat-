# 快速开始指南

## 项目概述

科讯嘉联训机师学苑积分管理平台是一个基于微信小程序和云开发的完整积分管理系统。

## 功能模块

### 学员端
- 🏠 首页 - 查看积分、等级、公告和热门商品
- 🛒 积分商城 - 浏览和兑换商品
- 📊 积分明细 - 查看积分变动记录
- 📦 兑换记录 - 查看订单状态
- 👤 个人中心 - 管理个人信息和收货地址

### 管理端
- ➕ 发放积分 - 给学员发放积分
- ✅ 核销订单 - 扫码或输入兑换码核销
- 👥 学员管理 - 查看学员信息
- 🛍️ 商品管理 - 管理商城商品

## 等级体系

| 等级 | 名称 | 积分要求 |
|------|------|----------|
| 🌱 初级训机师 | 0 - 999 积分 | 基础权益 |
| ⭐ 中级训机师 | 1000 - 4999 积分 | 专属折扣 |
| 👑 高级训机师 | 5000+ 积分 | VIP专属 |

## 部署步骤

### 第一步：环境准备

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号，获取 AppID
3. 在微信公众平台开通云开发

### 第二步：导入项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择项目目录：`kxjl-points-mall`
4. 填入 AppID 和项目名称
5. 点击「导入」

### 第三步：开通云开发

1. 在微信开发者工具中，点击工具栏的「云开发」按钮
2. 点击「开通」，按照提示完成开通
3. 记录云开发环境 ID

### 第四步：配置环境

打开 `miniprogram/app.js`，修改云环境 ID：

```javascript
wx.cloud.init({
  env: 'your-env-id', // 替换为你的云开发环境ID
  traceUser: true
});
```

### 第五步：创建数据库

参照 `cloudbase/init-database.md` 文档：

1. 创建所有需要的数据库集合
2. 设置集合权限
3. 导入初始数据（等级、分类、示例商品等）
4. 添加管理员账号

### 第六步：部署云函数

1. 在左侧文件目录中，找到 `cloudfunctions` 文件夹
2. 右键点击每个云函数文件夹（login、getPoints、exchange、adminGrantPoints、verifyOrder）
3. 选择「上传并部署：云端安装依赖」
4. 等待所有云函数部署完成

### 第七步：测试运行

1. 点击「编译」按钮
2. 在模拟器中测试各项功能
3. 使用真机扫码测试

## 常见问题

### Q: 云函数部署失败怎么办？

A: 检查以下几点：
1. 确认云开发环境已开通
2. 检查网络连接
3. 尝试右键云函数文件夹，选择「在终端中打开」，手动运行 `npm install`

### Q: 数据库权限如何设置？

A: 参考 `cloudbase/init-database.md` 文档中的权限设置说明。

### Q: 如何添加管理员？

A: 在云开发控制台的 `admins` 集合中添加一条记录，填入你的 openid。

### Q: 如何获取 openid？

A: 先在小程序中登录一次，然后在云开发控制台的 `users` 集合中找到你的记录，复制 `_openid` 字段。
