# 科讯嘉联训机师学院积分管理平台

基于微信小程序 + 云开发的积分管理平台，用于学员积分管理、等级划分和积分商城兑换。

## 功能特性

- 学员端功能
  - 积分查询与明细
  - 等级展示与升级
  - 积分商城商品浏览
  - 商品兑换
  - 兑换记录查询
  - 个人中心

- 管理端功能
  - 积分发放
  - 订单核销
  - 学员管理

## 技术栈

- 前端：微信小程序原生开发
- 后端：微信云开发（云数据库、云函数、云存储）

## 等级体系

| 等级 | 名称 | 积分要求 |
|------|------|----------|
| junior | 初级训机师 | 0 - 999 |
| intermediate | 中级训机师 | 1000 - 4999 |
| senior | 高级训机师 | 5000+ |

## 项目结构

```
kxjl-points-mall/
├── miniprogram/              # 小程序前端
│   ├── pages/                # 页面
│   ├── components/           # 组件
│   ├── utils/                # 工具函数
│   ├── app.js
│   ├── app.json
│   └── app.wxss
├── cloudfunctions/           # 云函数
├── cloudbase/                # 云数据库初始化
├── docs/                     # 文档
└── project.config.json       # 小程序配置
```

## 快速开始

1. 在微信开发者工具中导入项目
2. 开通云开发环境
3. 部署云函数
4. 初始化数据库
5. 编译运行
